SpeechTest is a demo project showing the iOS 7 speech synthesis functionalities of U3DXT.
It speaks the phrases you enter using iOS 7's new speech synthesizer.
You can also change the voice's language, pitch, speed, and volume.

There are a few steps you need to take to get this working.

Set Up Project in Unity 4:
1. In menus, U3DXT->Select Modules..., select "Speech Synthesis for iOS 7".
2. Build with XCode 5 and run on iOS 7 device.
