IAPTest is a demo scene showing the native in-app purchase functionalities of U3DXT.

There are a few steps you need to take to get this working.

1. Setup In-App Purchase on iTunesConnect for your game.
2. Change all references of "com.vitapoly.gamekittest" to your game's.
3. Make sure your bundle ID is the same as what you set up in iTunesConnect.
4. Build and run on device.
